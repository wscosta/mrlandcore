# correctGAEZv4

Correct Global Agro-ecological Zones (GAEZ) data

## Usage

``` r
correctGAEZv4(x)
```

## Arguments

- x:

  MAgPIE object provided by readGAEZv4 function

## Value

MAgPIE object at 0.5 cellular level

Felicitas Beier

## Examples

``` r
if (FALSE) { # \dontrun{
readSource("GAEZv4", convert = "onlycorrect")
} # }
```
