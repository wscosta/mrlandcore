# correctLPJmLClimateInput_new

Correct LPJmL climate input variables

## Usage

``` r
correctLPJmLClimateInput_new(x)
```

## Arguments

- x:

  magpie object provided by the read function

## Value

Magpie objects with results on cellular level, weight, unit and
description.
