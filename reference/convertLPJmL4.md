# convertLPJmL4

Convert LPJmL content

## Usage

``` r
convertLPJmL4(x)
```

## Arguments

- x:

  magpie object provided by the read function

## Value

List of magpie objects with results on cellular level, weight, unit and
description.

## See also

\[readLPJmL()\]

``` r
if (FALSE) { # \dontrun{
readSource("LPJmL4", subtype = "soilc", convert = TRUE)
} # }
```
