# readLPJmL4

Read LPJmL content

## Usage

``` r
readLPJmL4(subtype = "LPJmL5:CRU4p02.soilc")
```

## Arguments

- subtype:

  Switch between different input

## Value

List of magpie objects with results on cellular level, weight, unit and
description.
