# correctLUH2v2

Correct LUH2v2 content

## Usage

``` r
correctLUH2v2(x, subtype)
```

## Arguments

- x:

  magpie object provided by the read function

- subtype:

  switch between different inputs

## Value

List of magpie object with results on cellular level
