# readLPJmLInputs

This function reads in LPJmL inputs (inputs to LPJmL)

## Usage

``` r
readLPJmLInputs(subtype = "lakeshare")
```

## Arguments

- subtype:

  Switch between different inputs

## Value

List of magpie objects with results on cellular level, weight, unit and
description.
