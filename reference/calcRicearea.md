# calcRicearea

calculates rice area based on LUH flooded areas and physical rice areas
reported by FAOSTAT.

## Usage

``` r
calcRicearea(cellular = FALSE, share = TRUE)
```

## Arguments

- cellular:

  If TRUE: calculates cellular rice area

- share:

  If TRUE: non-flooded share is returned. If FALSE: rice area (flooded,
  non-flooded, total) in Mha is returned

## Value

rice areas or rice area shares of flooded and non-flooded category
