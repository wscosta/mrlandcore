# correctLandInG

correct LandInG data. Convert unit from ha to mio ha

## Usage

``` r
correctLandInG(x)
```

## Arguments

- x:

  magpie object provided by the read function

## Value

corrected magpie object

## See also

[`readLandInG`](readLandInG.md)

``` r
if (FALSE) { # \dontrun{
```
