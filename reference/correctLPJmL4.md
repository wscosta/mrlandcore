# correctLPJmL4

Correct LPJmL4 content

## Usage

``` r
correctLPJmL4(x)
```

## Arguments

- x:

  magpie object provided by the read function

## Value

List of magpie objects with results on cellular level, weight, unit and
description.

## See also

\[correctLPJmL4()\]

Kristine Karstens, Felicitas Beier

## Examples

``` r
if (FALSE) { # \dontrun{
readSource("LPJmL4", subtype = "soilc", convert = "onlycorrect")
} # }
```
